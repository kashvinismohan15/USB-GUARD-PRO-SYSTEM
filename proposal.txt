1. Background of the Study 
In recent years, the use of Universal Serial Bus (USB) devices has become 
widespread due to their portability and convenience. However, this convenience 
also introduces significant cybersecurity risks. Malicious actors exploit USB 
devices to deliver malware, launch keylogging attacks, and even take control of 
computer systems using techniques such as BadUSB and HID (Human 
Interface Device) injection. Many existing antivirus programs only scan the file 
content within a USB drive but fail to detect unusual device behavior. 
This project, titled USBGuard Pro: Smart USB Intrusion Prevention System, 
aims to develop a lightweight, intelligent software that monitors USB device 
activities and detects suspicious behavior patterns in real time. By identifying 
and preventing unauthorized USB actions, this system enhances endpoint 
security and provides users with better protection against physical intrusion 
threats. 
2.  Problem Statement 
USB devices are one of the most common vectors for cyberattacks because 
they can bypass traditional network defenses. Attackers can easily disguise a 
USB device as a keyboard or network adapter, enabling data theft, malware 
infection, or remote system access. Existing security tools mainly focus on 
scanning files stored on the USB, leaving behavioral threats undetected. 
Organizations and individual users are increasingly vulnerable to such threats 
due to the absence of an intelligent mechanism that monitors USB behavior at 
the system level. Therefore, there is a need for an automated intrusion 
prevention system that can detect, alert, and block abnormal USB behavior in 
real time.

3. Objective 
• To study USB-based attack methods and existing countermeasures. 
• To develop a lightweight USB intrusion prevention system that detects and 
blocks unauthorized USB behavior. 
• To test the system functionality and robustness of the proposed intrusion 
detection system. 
4. Scope 
This project focuses on monitoring USB connections and detecting malicious 
behavior patterns on personal computers or laptops. It will identify unauthorized 
USB devices, HID-based attacks, and data exfiltration attempts. 
Target Users: 
• IT administrators and general computer users. 
System Functions: 
• Detects new USB connections and verifies device identity. 
• Monitors USB behavior for unusual patterns. 
• Alerts users of suspicious activity. 
• Blocks or quarantines malicious devices. 
Platform: 
Windows or Linux desktop environment using Python and PyUSB libraries.

5. Significant of the Project 
USBGuard Pro offers an innovative approach to endpoint security by focusing on real-time 
behavior monitoring rather than traditional file scanning. This project will raise awareness 
about physical device-based threats and provide a cost-effective, user-friendly solution to 
prevent USB intrusion. It will also benefit small businesses, institutions, and individuals by 
reducing data leakage and unauthorized access caused by infected USB devices. 
6. Literature Review 
USB-based attacks have been a consistent cybersecurity concern for over a 
decade. According to Kaspersky (2023), USB malware infections increased by 17% 
in 2022 compared to the previous year, emphasizing the growing threat of physical 
attack vectors. Research by Wang and Stavrou (2010) demonstrated the feasibility 
of BadUSB attacks, where firmware manipulation allows a USB device to 
masquerade as a keyboard. Similarly, tools like USB Rubber Ducky have made 
HID injection attacks accessible to the public. 
Several defensive tools exist, such as USB Kill Switch and endpoint protection 
software, but they are often complex or resource intensive. Therefore, a lightweight 
solution focused on behavior detection rather than signature-based scanning is 
crucial. This project draws inspiration from behavioral intrusion prevention models 
and aims to implement a user-friendly and effective defense against USB attacks.

7. Methodology 
Development Model: 
Waterfall Model 
Phases: 
1. Requirement Analysis – Identify USB attack patterns and required features. 
2. System Design – Create architecture for monitoring, detection, and alerting 
components. 
3. Implementation – Develop the system using Python and PyUSB. 
4. Testing – Evaluate accuracy, performance, and usability. 
5. Deployment – Run real-world tests with multiple USB devices. 
Hardware & Software Requirements: 
• Hardware: Standard laptop (Intel i5+, 8GB RAM) 
• OS: Windows/Linux 
• Backend: Python with PyUSB library 
• Database: SQLite (for logging) 
• GUI: Tkinter or PyQt 
• IDE: VS Code / PyCharm 
Estimated Budget: 
• Development and testing tools – RM100 
• USB devices for testing – RM50 
Total Estimated Cost: ~RM150 
Conclusion 
USBGuard Pro represents a new generation of lightweight cybersecurity tools designed to 
protect systems from physical intrusion through USB devices. By focusing on behavioral 
monitoring, the project aims to detect attacks that traditional antivirus systems cannot. The 
outcome will contribute to safer computing environments and enhance awareness of physical
layer cybersecurity threats.

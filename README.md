# USBGuard Pro: Smart USB Intrusion Prevention System

USBGuard Pro is a cybersecurity solution designed to detect, monitor, and prevent unauthorized USB device activities on Windows systems. The system provides real-time USB event monitoring, device logging, and intrusion prevention mechanisms to protect computers from potential threats introduced through removable storage devices.

This project aims to enhance endpoint security by identifying abnormal USB behaviors, maintaining detailed activity logs, and allowing only trusted USB devices to access the system. Developed using Visual Studio Code on the Windows platform, USBGuard Pro offers a lightweight and user-friendly approach to USB security management.

## Key Features

* Real-time USB device detection and monitoring
* USB connection and disconnection event logging
* Trusted and untrusted USB device identification
* Detection of suspicious USB activities
* Automatic prevention of unauthorized USB access
* User-friendly interface for monitoring USB events
* Secure log generation for forensic analysis

## Technologies Used

* Python
* Windows Operating System
* Visual Studio Code
* USB Device Monitoring APIs
* File Logging and Event Management

## Project Objectives

* Enhance system security against USB-based threats
* Monitor and record USB device activities
* Detect abnormal USB behaviors
* Prevent unauthorized data access through removable devices
* Provide security administrators with actionable USB activity insights

This project was developed as a Final Year Project (FYP) focusing on endpoint security and USB threat mitigation in modern computing environments.
